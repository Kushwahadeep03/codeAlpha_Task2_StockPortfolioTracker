# Stock Portfolio Tracker

## Project Goal
This project is a simple Stock Portfolio Tracker that calculates the total investment based on manually defined stock prices.

## Features
- User can input stock names and quantities.
- Stock prices are stored in a dictionary.
- Calculates total investment for all stocks added.
- Displays a portfolio summary.

## How It Works
1. The program shows a list of available stocks and their prices.
2. User enters the number of different stocks they want to add.
3. For each stock, user inputs:
   - Stock name (validated against the available stocks)
   - Quantity
4. The program calculates the cost for each stock and adds it to the total investment.
5. After all entries, it displays a portfolio summary with individual stock values and the total investment.
